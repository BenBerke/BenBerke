# Hi, I'm Berke Memioglu

Computer Science student at Newcastle University focused on C/C++ systems programming, graphics, game engine tooling, and low-level software.

My main project is [Tilky Engine](https://github.com/BenBerke/Tilky-Engine), an open-source sector-based 3D game engine and editor built with C++20, OpenGL, SDL3, Lua, OpenAL, ImGui, Tracy, and Rust tooling.

I also build CPU renderers, interpreters, virtual machines, digital logic experiments, and performance-focused tools.
